Splatoon 3 Weapon Randomizer

This is a highly customizable website that allows for users to randomly select their weapons for Splatoon 3

The goal of this website is to allow streamers to have a fun and endgaging way to display the weapon they will be playing

This website is not affiliated with Nintendo or any other company, all images, sounds fonts and resources belong to Nintendo.
